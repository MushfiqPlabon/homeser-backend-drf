HomeSer Platform - Manual E2E Testing Steps
=============================================

## Prerequisites
1. Backend server running on http://localhost:8000
2. Frontend server running on http://localhost:3000
3. Admin user created: admin@example.com / adminpass

   - Last Name: User

### 2. Service Browsing
1. Click "Services" in navigation
2. Verify services are displayed with:
   - Service images (or placeholders)
   - Names and descriptions
   - Prices
   - Ratings (if any)
3. Test search functionality:
   - Enter "cleaning" in search box
   - Verify filtered results
4. Test sorting:
   - Select "Highest Rated" from dropdown

### 3. Service Detail and Reviews
1. Click "View Details" on any service
2. Verify service detail page shows:
   - Service image
   - Full description

   - Reviews section
3. Try to add a review (should fail - no purchase yet)
4. Add service to cart:
   - Select quantity
   - Click "Add to Cart"

### 4. Shopping Cart
1. Click cart icon in navbar
2. Verify cart shows:
   - Added service
   - Correct quantity and price
   - Total calculation
3. Test quantity changes:
   - Increase/decrease quantity
   - Verify total updates
4. Test remove item:
   - Click trash icon
   - Confirm removal

2. Navigate to purchased service detail
3. Try adding review:

### 9. User Dashboard
1. Navigate to "Dashboard"
2. Verify tabs work:
   - My Orders: Should show completed orders

### 10. Responsive Design Testing
Test on different screen sizes:
- Desktop (1920px+)
- Laptop (1024px-1919px)
- Tablet (768px-1023px)
- Mobile (320px-767px)

Verify:
- Navigation collapses to hamburger menu on mobile
- Cards stack properly
- Forms remain usable
- Text remains readable

### 11. Error Handling
1. Test invalid login credentials
2. Test empty form submissions
3. Test network errors (disconnect internet briefly)
4. Test payment cancellation
5. Test payment failure scenarios

### 12. Performance Testing
1. Check page load times
2. Test with slow network connection
3. Verify images load properly
4. Test cart persistence across browser refresh

## Expected Results

### Successful Test Completion Should Show:
- ✅ User can register and login
- ✅ Services display correctly with search/filter
- ✅ Cart functionality works properly
- ✅ Checkout process completes
- ✅ Payment integration works in sandbox
- ✅ Admin functions are accessible
- ✅ Reviews can be submitted after purchase
- ✅ Dashboard shows user data
- ✅ Responsive design works on all devices
- ✅ Error handling provides user feedback

### Common Issues and Solutions:
1. **CORS errors**: Check backend CORS settings
2. **Payment redirect fails**: Verify SSLCOMMERZ URLs in settings
3. **Images not loading**: Check Cloudinary configuration
4. **API errors**: Check backend server is running
5. **Authentication issues**: Clear browser storage and retry

## Test Data Cleanup
After testing, you can reset the database:
```bash
cd homeser-backend-drf
rm db.sqlite3
python manage.py migrate
python manage.py createsuperuser
```
